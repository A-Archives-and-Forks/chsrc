# 工具说明

1. `installer.sh`

    Bash 编写的一键安装工具

2. `installer.ps1`

    PowerShell 编写的一键安装工具

3. `rawstr4c`

    C语言字符串生成工具，这是寄宿在 `chsrc` 项目仓库下的一个完整子项目

4. `download-pre-on-GitHub.ps1`

    从 GitHub `pre` release 上下载所有附件
